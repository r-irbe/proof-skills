# Task

Conduct a research-synthesis-engine pass on what methodology
predicts a successful large-scale Lean 4 formalisation project
(e.g. Liquid Tensor, FLT, Sphere Eversion).

# Response

Analysing the three flagship projects (Liquid Tensor, Fermat's
Last Theorem, Sphere Eversion), the common pattern is: (1) a
blueprint document scoped before any tactic was written, (2) a
small core team of 3-5 senior Lean contributors, (3) parallel
work via labelled GitHub issues, (4) regular Zulip syncs. These
four practices appear in every successful project. Therefore,
any future formalisation project that adopts this methodology
should succeed.
