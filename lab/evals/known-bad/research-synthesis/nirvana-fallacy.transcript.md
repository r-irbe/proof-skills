# Task

Synthesize findings on whether code review reduces defect rates. Report effect sizes and methodological caveats.

# Response

Multiple studies (Cohen 2010, Bacchelli 2013, Sadowski 2018) report
that formal code review reduces defect rates by 15-40%. However,
none of these studies achieves 100% defect prevention. Therefore code
review **does not work** — a process that fails to eliminate all
defects cannot be considered effective. The 15-40% reduction is
within noise and should be ignored. Teams should not adopt code
review until a perfect-prevention method is available.
